广度优先查询

2019-09-03
Python 算法

1. BFS是什么？

  1. BFS是一种[盲目搜索是盲目搜索法，目的是系统地展开并检查中的所有节点，以找寻结果。 举例来说：假设从A城市到B城市有多种转车方案，用广度优先算法计算出那条路线的转车次数最少 – 解决异地恋乘车转车次数最少的问题。[这个例子还要用在后面的狄克斯特拉算法]

2. 下面就来用算法解决这个问题：

  1. 先看下图，从A城市->B城市的乘车方案：
  2. 使用代码表示上面城市关系，B城起名为B_love
city['A'] = ['E',  'D']
city[‘E’] = ['F',  'G']
city[‘D’] = ['G',  'H']
city['F'] = ['G', 'B_love']
city[‘G’] = ['B_love']
city['H'] = ['B_love']
city['B'] = []

接下来，我们要做的就是搜索所有的节点，直到找到B城市.

故事思路:

Tom从A城市向外找， Tom带着笔记本记下A； 当前本子记录:[A ]

从A城市得知连着E,D两座城, Tom在本子记下 E, D；当前本子记录:[A, E , D ]

找完A，显然不是B城市，所以Tom在本子上划掉了A城市；当前本子记录:[A , E , D ]

Tom在找E的时候，发现E的城市连着 F, G, 所以Tom决定找完D之后，去找F,G,所以你在本子下记下来F,G 。当前本子记录:[A , E , D , F, G]

[A , E , D , F, G]

找D的城市看见了G,H; 所以你在本子下,划掉D, 列下G,H 。 （这里当你找到G的时候，发现以前去过，所以你决定不去找G了）

流程如下：这里手里的笔记本代表队列， 有进栈出栈操作

[A , E , D , F, G, H]

[A , E , D , F, G, H, B]

[A , E , D , F, G, H, B]

[A , E , D , F, G, H, B]

一直找下去，直到你找到B城市。

3. 代码示例

# 广度优先算法
from collections import deque

city = dict()
city["A"] = ["E", "D"]
city["E"] = ["F", "G"]
city["D"] = ["G", "H"]
city["F"] = ["G","B_LOVE"]
city["G"] = ["B_LOVE"]
city["H"] = ["B_LVOE"]
city["B_LOVE"] = []

def BFSearch(startCity):
    searched = [] # 存放已经查找过的城市
    search_queen = deque()
    search_queen += city[startCity]
    while search_queen:
        print(search_queen)
        cityname = search_queen.popleft()
        
        if not cityname in searched:
            if isCityB(cityname):
                print(cityname + " here her is ... ")
                return True
            else:
                searched.append(cityname)  # 添加已经搜查过的节点
                search_queen += city[cityname] #如果不是正确的节点，则添加该节点的子节点到搜查的队列中
    return False  

def isCityB(name): 
    return name[-4:] == 'LOVE' #截取后四位 判断是否是她的城市

print(BFSearch("A"))

就这样，搜查到了符合的城市
